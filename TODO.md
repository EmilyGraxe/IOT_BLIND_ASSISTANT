# SMARTASSIST — Project Planning & Backlog

## Project Title
SMARTASSIST: IoT Hybrid Navigation System for Visually Impaired Persons

---

# Project Goal

To develop an IoT-based assistive navigation system that improves mobility and safety for visually impaired individuals using ESP32, ultrasonic sensors, vibration feedback, and cloud notifications.

# Challenges Encountered

## Challenge 1
Sensor interference caused inconsistent readings.

### Solution
Implemented sequential sensor reading and timing delays.

---

## Challenge 2
WiFi connectivity instability during testing.

### Solution
Optimized Blynk connection handling and improved power stability.

---

## Challenge 3
False obstacle detection from environmental reflections.

### Solution
Adjusted sensor thresholds and calibrated detection distances.
